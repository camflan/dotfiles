# Example Skill

This is an example skill demonstrating the basic structure.

## When to Use

Use this as a template when creating new custom skills.

## Instructions

1. Copy this directory structure for new skills
2. Update the frontmatter with your skill name and description
3. Write clear, step-by-step instructions
4. Keep the SKILL.md file under 500 lines

## File Structure

```
skill-name/
└── SKILL.md (this file)
```

For complex skills, you can add:
- `reference.md` - detailed reference docs
- `examples.md` - usage examples
- `scripts/` - helper scripts
